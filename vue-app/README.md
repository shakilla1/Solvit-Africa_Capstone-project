# Vue Frontend

This folder contains the Vue frontend for the PhishGuard Pro project.

## What it includes

- Vue 3 + Vite client application
- Admin and user pages for campaign management and phishing awareness training
- Reusable components, routing, and store management

## How to run

1. Open a terminal in this folder:
   ```bash
   cd "Build Functional Laptop Design/vue-app"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

The Vue app runs on `http://localhost:5173` by default and expects the backend API at `http://localhost:8080`.
